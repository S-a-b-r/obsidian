# Template в компоненте(шаблон)

Шаблон может включать в себя:

1. Интерполяции:
    - Текст.
    - Сырой HTML.
    - Атрибуты.
    - Выражения JavaScript.
2. Директивы:
    - Аргументы.
    - Динамические аргументы.
    - Модификаторы.
3. Сокращения:
    - Сокращение v-bind.
    - Сокращение v-on.
