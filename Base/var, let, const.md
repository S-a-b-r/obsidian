# var, let, const
Почему нельзя объявлять переменную так
`a = 1` - она попадает в глобальную область видимости.
Пример:
```
function foo() {
	a = 1;	// `a` формально не объявлена
}

foo();
a;			// 1 — упс, автоматическая глобальная переменная :(
```
