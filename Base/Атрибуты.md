# Атрибуты
Некоторые директивы могут принимать «аргумент», отделённый от названия директивы двоеточием. Например:
```
<a v-bind:href="url"> ... </a>
```

`href` — аргумент, указывающий директиве `v-bind` связать атрибут `href` элемента со значением выражения `url`.
